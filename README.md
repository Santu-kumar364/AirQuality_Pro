# 🌍 Real-time Air Quality Monitoring Dashboard

🚀 **Live Demo:** http://air-quality-dashboard-santu.s3-website.ap-south-1.amazonaws.com/  
💻 **Tech:** React.js · Python · AWS Lambda · API Gateway  

A scalable full-stack application for real-time air quality monitoring using a serverless architecture.

---

## 🚀 Features

- 📊 Real-time environmental data visualization  
- ⚡ Low-latency backend using AWS Lambda + API Gateway  
- 👥 Supports 500+ users  
- 📈 High availability system (99.8% uptime)  
- 📱 Responsive UI with dynamic charts  

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Python (AWS Lambda - Serverless)  
- **APIs:** REST APIs, API Gateway  
- **Cloud:** AWS (Lambda, API Gateway, S3)  

---

## 🧠 System Architecture

Client (React) → API Gateway → AWS Lambda → Data Source  

✔ Serverless architecture ensures scalability and cost efficiency
